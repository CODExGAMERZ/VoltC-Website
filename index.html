The Evolution: VoltC v1 vs v2

VoltC v2 is a massive upgrade across every layer, introducing full Git support, deep AI assistance, and professional PTY persistent terminal streams.

Feature VoltC v1 VoltC v2 Stands Out
Git Integration × None ✓ Stage/commit/diff changes directly inside sidebar drawer, branch switches, push/pull
Terminal Stream ✓ Sandbox WebSocket terminal ✓ Persistent PTY (resizes, reconnects, process history buffer)
Memory Visualizer ✓ Static Stack visualizer ✓ Dynamic Stack + Heap representation with interactive pointer indicators
Themes System Midnight Crimson only ✓ 5 presets (Midnight, Void, Solar, Arctic, Hacker) + Appearance customizer
Compiler Sanitizers × Standard flag presets ✓ Automated Address/Undefined sanitizer toggle (-fsanitize=address,undefined)
C Starter Templates 2 files ✓ 12 curated templates for beginners to advanced structures (Makefiles, structs, linked lists)

Core Architecture Features

VoltC is tailored for local speed and visual understanding, making it the perfect platform for students and expert C developers alike.

Git Integration

Manage code versions without opening a terminal. View untracked and modified files, stage changes, view syntax-highlighted diffs, and write commit messages in the panel.

branch: main
[M] buggy.c (unstaged changes)

Dynamic Memory Visualizer

Understand pointer variables and dynamic allocations instantly. Renders heap variables, stack boundaries, and links them with live arrow structures during compilation builds.

int *ptr
Addr: 0x7ffd5

Sandboxed Compilation

Compiles in isolated workspace boundaries with a default 10s subprocess timeout that stops infinite loop freezes before they brick your Linux desktop resources.

[TIMEOUT ERROR] Process terminated.
Install VoltC on Ubuntu

Choose your preferred installation method. The stable snap includes all environment plugins boxed out of the bag.

terminal
Ubuntu Shell
$ sudo snap install voltc
# VoltC will auto-connect plugs for network, home and X11 graphic outputs.

sudo dpkg -i voltc_2.0_amd64.deb
sudo apt-get install -f
$ git clone https://github.com/CODExGAMERZ/VoltC.git
cd VoltC && git checkout v2
pip3 install -r requirements.txt
